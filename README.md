### 🔨 Building now

**[ModForge](https://github.com/champmk/modforge)** — a deterministic migration engine for Minecraft mods.
When the game renames a class, method, or field across versions, ModForge resolves the new name from
**real mappings and the real game jars**, labeling every answer `EXACT`, `CANDIDATE`, or `UNRESOLVED`
instead of hallucinating one. Ships an **MCP server** so AI coding agents can query it for version-truth.

Install: `npm i -g modforge` · [npm](https://www.npmjs.com/package/modforge) · MIT · zero runtime deps
